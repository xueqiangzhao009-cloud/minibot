# Summarize

Summarize text and documents to get the key points quickly.

## Usage

### Summarize text
```
Summarize this text: "The quick brown fox jumps over the lazy dog. It was a beautiful day, and the fox was feeling energetic."

Summarize the following article: [paste long text here]
```

### Summarize a file
```
Summarize the file example.txt
Summarize the document report.pdf
```

### Summarize a webpage
```
Summarize the webpage https://example.com
Summarize the article at https://example.com/article
```

## Examples

- "Summarize the Gettysburg Address"
- "Summarize the file README.md"
- "Summarize the webpage https://en.wikipedia.org/wiki/Artificial_intelligence"
