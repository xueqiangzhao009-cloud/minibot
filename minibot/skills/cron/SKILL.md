# Cron

Schedule one-time or recurring tasks.

## Usage

### One-time reminder
```
Remind me to call Alice at 3pm
Schedule a task to check email at 9:30 tomorrow morning
```

### Recurring tasks
```
Remind me to take a break every hour
Schedule a daily task to backup files at 10pm
```

### List scheduled tasks
```
List all scheduled tasks
Show my reminders
```

### Cancel tasks
```
Cancel the task to call Alice
Remove the daily backup task
```

## Examples

- "Remind me to buy milk when I get home"
- "Schedule a task to check the weather every morning at 8am"
- "Cancel all my reminders"
