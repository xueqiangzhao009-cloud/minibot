# GitHub

Interact with GitHub repositories, including cloning, committing, and pushing changes.

## Usage

### Clone a repository
```
Clone the repository https://github.com/minibot-ai/minibot.git
Clone the repo minibot-ai/minibot to the current directory
```

### Commit changes
```
Commit changes with message "Add new feature"
Commit all changes with message "Update documentation"
```

### Push changes
```
Push changes to origin
Push to the main branch
```

### Pull changes
```
Pull latest changes from origin
Pull from the develop branch
```

### Check status
```
Check git status
Show current branch
```

- "Clone the repository https://github.com/openai/openai-python.git"
- "Commit changes with message 'Fix bug in API server'"
