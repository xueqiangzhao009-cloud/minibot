# Calculator

```
Calculate 2 + 2
What's 10 * 5?

```

### Complex calculations
```
Calculate 2^10
What's the square root of 144?
Calculate 3.14 * 5^2
What's 10! (factorial)
```

Convert 100 degrees Celsius to Fahrenheit
