# Memory

Store and retrieve information for later use.

## Usage

### Store information
```
Remember that my favorite color is blue
Store that the meeting is at 2pm tomorrow
Save the fact that the API key is "sk-1234567890"
```

### Retrieve information
```
What's my favorite color?
When is the meeting?
What's the API key?
```

### List stored information
```
List all stored information
Show me what I've saved
```

### Delete information
```
Forget about my favorite color
Delete the meeting information
Clear all stored information
```

## Examples

- "Remember that my birthday is on June 15"
- "What's my birthday?"
- "Forget about my birthday"
